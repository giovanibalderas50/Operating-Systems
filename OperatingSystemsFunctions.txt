One function of the operating system is to boot the computer. Also known as starting a computer. It checks the computer and makes sure it is ready to work.
Another function of the operating sysetm is to control all devices attached to the computer. The hardware devices are controlled with the help of small software called device drivers.which is pretty cool if you ask me.
Of the many things an operating system does, it also secures data. It protects data stored on the computer from any modification or deletion. Also is some cases illegal use.
